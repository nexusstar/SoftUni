## Using Neovim to edit JS

### Setup Neomake to lint js files

* first install [ JSCS ](http://jscs.info/)

```bash
    npm install -g jscs
```
* (optional) generate .jscsrc for the project

```bash
    jscs --auto-configure ./
```
